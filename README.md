# AI Placement Intelligence

Predicting Placement Success, Salary and Employability Risk using Machine Learning

---

## Overview

AI Placement Intelligence is a machine learning-based system designed to predict:

- Placement probability (within 3/6/12 months)
- Expected salary range
- Employability risk level (Low / Medium / High)

This system helps students, institutions, and education loan providers make data-driven decisions.

---

## Problem Statement

Education loan providers and institutions face challenges in:

- Predicting placement timelines  
- Estimating salary packages  
- Identifying students at risk of delayed placement  
- Providing early intervention support  

---

## Solution

This system uses machine learning to:

- Analyze student, institute, and market data  
- Predict placement outcomes  
- Classify risk levels  
- Provide actionable insights  

---

## Features

### Student Features
- CGPA  
- Internship Duration  
- Course Type  

### Institute Features
- College Tier  
- Placement History  

### Market Features
- Hiring Trends  
- Market Conditions  

---

## Machine Learning Model

- Model: Random Forest  
- Reason for selection:
  - Handles non-linear data
  - High accuracy
  - Reduces overfitting
